# IVI-NSCLC
`iviNSCLC` is an R package that runs the [Innovation and Value Initiative's (IVI's)](http://www.thevalueinitiative.org/)  non-small cell lung cancer (NSCLC) simulation model (the IVI-NSCLC model). The models simulates the costs, health outcomes, and risks associated with sequences of treatment including EGFR Tyrosine Kinase Inhibitors (TKIs), platinum-based doublet chemotherapy (PBDC), antiâ€“vascular endothelial growth factor (anti-VEGF) therapy, and immune checkpoint inhibitors for patients with epidermal growth factor receptor (EGFR) positive NSCLC.
